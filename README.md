# Поиск изображений по текстовому запросу

## О проекте
Демонстрационная версия поиска изображений по текстовому описанию.

## Используемые технологии
- Python
- PyTorch, TensorFlow
- BERT, ResNet50, CLIP
- CatBoost, LightGBM

## Результаты
Лучшая модель: CLIP (R² = 0.1235)

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите Jupyter Notebook: `jupyter notebook project.ipynb`
