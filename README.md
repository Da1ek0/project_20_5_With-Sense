# Проект 5: Поиск фото по описанию
# О чём проект
Мы делаем демо-версию для фотохостинга «Со Смыслом». Это сервис, где фотографы выкладывают снимки и добавляют к ним описание. Особенность — описывать фото могут не только авторы, но и другие пользователи.

Нужно показать начальству, что идея работает. Для этого сделаем PoC (проверку концепции) — упрощённую версию поиска картинок по текстовым запросам.

# Что нужно сделать
Обучить модель, которая:

- берёт вектор изображения и вектора текста

- выдаёт число от 0 до 1 — насколько они подходят друг другу

Это будет основа для будущего продукта.

# Ограничения по закону
В некоторых странах нельзя показывать фото детей (до 16 лет) без согласия родителей. В PoC мы не можем это проверять, поэтому просто удалим такие данные. А если в запросе встретится что-то "запрещённое" — покажем заглушку.

# Данные
Всё лежит по [ссылке](https://code.s3.yandex.net/datasets/dsplus_integrated_project_4.zip).

# Для обучения:

- train_dataset.csv — названия фото, id описания и сам текст (до 5 описаний на фото)

- train_images/ — сами фото

# Оценки от людей:

- CrowdAnnotations.tsv — результаты краудсорсинга (сколько людей сказали "да" или "нет")

- ExpertAnnotations.tsv — оценки трёх экспертов (от 1 до 4, где 1 — совсем не подходит, 4 — идеально)

# Для теста:

- test_queries.csv — запросы и id нужных фото

- test_images/ — тестовые фото
